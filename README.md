# Financial_Visualizer

Financial Visualizer is a full-stack Spring Boot application designed to simplify expense management. It features an AI-powered OCR engine that extracts total amounts from receipt images, eliminating the need for manual data entry.

Check out the live application here: https://financial-visualizer-1.onrender.com/

Features
Smart OCR Extraction: Uses Tesseract OCR to scan receipts and extract price data automatically.

Financial Dashboard: Visualize your spending habits with a clean and intuitive UI.

Persistent Storage: Integrated with PostgreSQL for secure and scalable data management.

Cloud Native: Fully containerized using Docker and deployed on Render.

Backend: Java 17, Spring Boot 3.2.2, Spring Data JPA

Database: PostgreSQL (Cloud)

OCR Engine: Tesseract (Tess4J)

DevOps: Docker, GitHub Actions, Render
